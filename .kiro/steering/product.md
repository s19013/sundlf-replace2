# Product Overview

SPA（シングルページアプリケーション）のWebアプリケーション。Laravel バックエンドと Vue.js フロントエンドを組み合わせた、認証機能を持つフルスタックWebサービス。

## Core Capabilities

- **ユーザー認証**: Laravel Sanctum を使用した SPA 認証（CSRF Cookie ベース）
- **REST API**: Scramble による OpenAPI 仕様自動生成、Ziggy によるルート共有
- **リアクティブUI**: Vue 3 + Pinia によるリアクティブな状態管理

## Target Use Cases

- 認証が必要な管理画面・ユーザー向けWebアプリケーションの開発基盤

## Value Proposition

- Laravel + Vue 3 のフルスタック構成で開発効率を最大化
- 型安全なAPI通信（OpenAPI → TypeScript スキーマ自動生成）
- Docker + mise による再現性の高い開発環境
